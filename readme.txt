Allows the user to present their personal or their project's Ohloh-metrics as a widget.

== Description ==

This plugin adds a new widget which can show the different available [Ohloh](http://www.ohloh.net/ "Ohloh, the open source network") widgets showing Ohloh metrics about a given project or information about the user's Kudo rank.

== Installation ==

1. Upload `ohloh-widget.php` to the `/wp-content/plugins/` directory
1. Place the widget into your sidebar using the 'Presentation' menu in WordPress
1. Adjust the widget's options so that it can show some useful content
1. Save changes. ;)
